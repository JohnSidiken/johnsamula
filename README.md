# John Samula - Portfolio Website

**Live Site:** https://johnsidiken.github.io/johnsamula/

### **About**
Personal portfolio showcasing my projects, skills, and contact info. Built with HTML, CSS, and vanilla JavaScript. Features dark/light mode, responsive design, and a working contact form.

### **Tech Stack**
- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox  
- **JavaScript** - Theme toggle, smooth scroll, form validation
- **GitHub Pages** - Hosting
- **FormSubmit** - Contact form backend

### **Features**
- Responsive design - works on mobile, tablet, desktop
- Dark/light theme toggle
- SEO optimized with meta tags and JSON-LD schema
- Accessible - WCAG compliant markup
- Fast load - no frameworks, minimal JS
